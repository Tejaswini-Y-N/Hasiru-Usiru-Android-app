# HASIRU–USIRU 🌱

## Smart Environmental Sustainability Android Application

Hasiru–Usiru is an Android-based environmental sustainability application developed using Kotlin and Firebase technologies. The application encourages eco-friendly participation by helping users monitor plantation activities, track oxygen contribution, and promote environmental awareness through an interactive digital platform.

---

## 📌 Features

- Plantation Activity Tracking
- Oxygen Contribution Monitoring
- Secure User Authentication
- Cloud Data Storage using Firebase
- Interactive User Interface
- Environmental Awareness System
- Responsive Android Application Design

---

## 🛠️ Technologies Used

- Kotlin
- XML
- Firebase Authentication
- Firebase Firestore
- Android Studio
- MVVM Architecture
- RecyclerView

---

## 🎯 Objectives

- Promote environmental sustainability and awareness.
- Encourage plantation participation through digital technology.
- Monitor oxygen contribution and eco-friendly activities.
- Develop a secure and scalable Android application.

---

## 📱 Application Modules

- User Authentication Module
- Dashboard Module
- Plantation Tracking Module
- Oxygen Monitoring Module
- Firebase Cloud Integration
- User Profile Management

---

## 🚀 Future Enhancements

- GPS-based plantation tracking
- AI-based environmental analytics
- Real-time air quality monitoring
- Community participation features
- Gamification and reward systems

---

## 📖 Conclusion

The Hasiru–Usiru project successfully combines Android application development with environmental sustainability concepts. The application improves user engagement in plantation activities while enhancing practical knowledge in Android development and Firebase integration.
